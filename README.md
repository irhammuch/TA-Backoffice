### Quick Start

#### 1. Clone repo

```
git clone https://github.com/irhammuch/TA-Backoffice.git
```

#### 2. Pindah ke direktori TA-Backoffice

```
cd TA-Backoffice
```

#### 3. Install Semua Dependencies

pakai npm:

```
npm install
```

atau pakai yarn:

```
yarn install
```

#### 4. Launch project

Development:

```
npm start
```

Production:

```
npm run start:prod
```
